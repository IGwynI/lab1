# Задание 3

## Задача 1

### Текст задачи

Опишите функции, осуществляющие основные математические операции над комплексными
числами (сложение, вычитание, умножение, деление, возведение в степень). Использовать
готовый класс Complex запрещено.

### Алгоритм решения

Пользователь вводит номер операции; 
Вводится действительная часть первого числа и его мнимая часть и действительная часть второго числа и его мнимая часть либо показатель степени для возвеления в степень;
